# Food Delivery Analytics Dashboard

## Project Overview

Designed an interactive Power BI dashboard to analyze a food delivery business using customer, restaurant, and order data.

The dashboard helps stakeholders monitor

• Revenue
• Orders
• Customers
• Restaurants

## Business Problem

The company wants to understand

- Why has revenue remained stable?
- Which customer segments contribute most to maintaining revenue?
- Are repeat customers helping keep revenue consistent?
- Which cuisines or restaurants consistently drive sales?

## Dataset

4 Tables

- Customers
- Restaurants
- Orders
- Date Table

Years Covered

2023–2025

## Data Cleaning

Cleaning was performed using Python (Pandas).

Tasks included

- Handling missing values
- Removing invalid IDs
- Standardizing categories
- Cleaning restaurant information
- Data validation

## Key Insights

- Revenue dropped in Q3 2024 despite similar order volume because cancelled orders increased and average order value declined.

- Weekend sales generate the highest revenue.

- Repeat customers contribute significantly to total revenue.

- A small number of restaurants contribute a large share of sales.
